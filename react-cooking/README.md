#运行npm install，安装依赖

#运行yarn start


#这是一个简单的菜谱项目，实现了菜单列表展示，菜单详情，搜索功能，其他功能还有待完善。主要是针对react-router的一个练习。

#项目调用了第三方数据平台的接口，因为接口调用有次数限制，所以可以自己到该平台注册申请一个appkey，https://www.jisuapi.com/。
